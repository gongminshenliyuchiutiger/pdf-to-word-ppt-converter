如何取得免費的 Google Gemini API 金鑰？

按照以下簡單步驟，即可取得您自己的免費金鑰：

  1. 前往 Google AI Studio 的 API 金鑰頁面。
  2. 使用您的 Google 帳號登入。
  3. 點擊頁面上的 「建立 API 金鑰」(Create API key) 按鈕。
  4. 系統會立即產生一組新的金鑰。點擊金鑰旁邊的複製圖示。
  5. 回到本頁面，將複製的金鑰貼到輸入框中即可！

提醒：請妥善保管您的 API 金鑰，不要與他人分享。
